Essay

Write an essay to discuss what a scale is and provide two ways it is useful in visualization

In D3, scales are functions that contain parameters that are defined by the user. These parameters are the domain and range. When these parameters have been defined, the scale function can be called with an input and it will return a scaled output value.

In one of the examples from the book, a version of the scale function is used in a scatterplot visualization. In this case, the scale is used to ensure that the correct values are getting the correct radius. Another useful way of visualization is creating domain and range axes, since it will produce an axis of numbers that can be inputed and an axis of numbers that will be outputted.
